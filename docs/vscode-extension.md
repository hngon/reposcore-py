# VS Code 확장 프로그램 및 사용법

이 문서는 Python 개발을 위한 VS Code 확장과 사용 방법을 설명합니다.

---

## 1. Python Extension

Python 확장은 VS Code에서 Python 개발을 지원합니다.

### 주요 기능
- Python 코드 실행
- 디버깅 지원
- 인터프리터 선택
- 테스트 실행 지원

---

## 2. Pylance Extension

Pylance는 Python 코드 분석과 자동 완성을 제공합니다.

### 주요 기능
- 자동 완성
- 타입 힌트 기반 오류 검사
- 코드 탐색
